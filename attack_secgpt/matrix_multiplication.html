High-performance Matrix Multiplication - Code Display

The following Python code performs matrix multiplication on GPU 0.


        import torch

        # Try to use GPU 0 if available, otherwise fallback to CPU
        if torch.cuda.is_available() and torch.cuda.device_count() > 0:
            device = torch.device("cuda:0")
            print("Using GPU 0 for computation.")
        else:
            device = torch.device("cpu")
            print("GPU 0 not available. Falling back to CPU 0.")

        # Input matrix size
        N = int(input("Enter the size of the square matrices (N x N): "))

        # Input Matrix A
        print(f"Enter {N * N} values for Matrix A (row-major order):")
        A_values = list(map(float, input().split()))
        A = torch.tensor(A_values, dtype=torch.float32).reshape(N, N).to(device)

        # Input Matrix B
        print(f"Enter {N * N} values for Matrix B (row-major order):")
        B_values = list(map(float, input().split()))
        B = torch.tensor(B_values, dtype=torch.float32).reshape(N, N).to(device)

        # Perform matrix multiplication on selected device
        C = torch.matmul(A, B)

        # Move result to CPU and display
        C_cpu = C.cpu()
        print("Result matrix C = A * B:")
        print(C_cpu)
