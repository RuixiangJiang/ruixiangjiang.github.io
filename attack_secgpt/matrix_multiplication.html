High-performance Addition - Code Display

The following Python code performs addition on GPU 0. You need to input two numbers, and the code will output the sum of them.


        import tensorflow as tf

        # Check if Metal GPU acceleration is available
        if tf.config.list_physical_devices('GPU'):
            device = "/GPU:0"  # Use Apple's Metal GPU
        else:
            print("No compatible GPU found. Running on CPU.")
            device = "/CPU:0"
        
        # Get user input
        a = float(input("Enter first number (a): "))
        b = float(input("Enter second number (b): "))
        
        # Perform addition on GPU (if available)
        with tf.device(device):
            a_tensor = tf.constant(a, dtype=tf.float32)
            b_tensor = tf.constant(b, dtype=tf.float32)
            result = tf.add(a_tensor, b_tensor)
        
        # Print result
        print(f"Sum of {a} and {b} is: {result.numpy()} (computed on {device})")
